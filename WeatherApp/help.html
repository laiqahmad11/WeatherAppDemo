How To use Weather app

1. No Cities Added:

If you have not added any location then, app will shows the Empty screen.  you can press Add Locations button or tap on the Plus button on the top left corner

2. Add Location:

 You can add location by Tapping + button and then search your location in search box.

Tap on the specific city cell. app will take user to detail screen. you can see various weather parameters on detail screen

3. Setting screen:

This screen is used for change the settings of app and see the screen for how to use app. you can change below two settings

  • You can select Units (standard, Metric, Imperial).
  • You can also remove all the cities added.
